Git Pull

If you're getting errors while pulling from a Git remote, follow these steps:

1. Check Git Remote

git remote -v

2. Add Remote If Missing

git remote add origin https://github.com/your-username/your-repo.git

3. List Remote Branches

git ls-remote --heads origin

4. Set Upstream Tracking

git branch --set-upstream-to=origin/master master

5. Pull With Rebase

git pull --rebase

Replace master with main if that's your remote branch name.
